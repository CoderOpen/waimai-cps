 ##### 安装后端
 * 原生php项目，php >= 5.6  
 * 直接指向根目录即可，/waimai.php文件中 根据代码注释修改网站接口域名 + 外卖项目配置信息,注意这里需要和前端请求地址保持一致
 
 ##### 安装前端
 * 修改/fornt/main.js 根据注释修改成个人信息
 * 使用uniapp 导入 front 目录进行编译安装发布上线即可，可参考官方文档 https://uniapp.dcloud.io/
